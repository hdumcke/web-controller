Instructions

Select a gait from the menu.

The start and stop buttons toggle the run mode.

When the robot is not running the sliders will set the values that are used by the robot when the start button is pressed.

When the robot is running the values of a slider will be sent to the robot.

When the stop button is pressed the robot will be set to the neutral position.

Tuning

For Walk, Trot and Gallop you can tune some internal parameters

Frequency: how fast each foot position is updated

Step Length: The length of the step

Step Height: The height used to calculate the Bezier curve for the swing phase

Number of Points: How many points are calculated for the stance and swing phases. Increasing the number of points will slow down the robot as more foot positions need updating
